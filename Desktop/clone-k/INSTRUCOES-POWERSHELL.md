# Instruções para PowerShell

## Passo 1: Instalar as dependências do preview

```powershell
npm install express cheerio jszip node-fetch
```

## Passo 2: Executar o servidor

```powershell
node preview-server.js
```

## Passo 3: Acessar no navegador

Abra: `http://localhost:3001`

---

## Comandos Rápidos (copiar e colar)

### Para instalar e rodar tudo de uma vez:

```powershell
npm install express cheerio jszip node-fetch && node preview-server.js
```

---

## Se quiser usar a versão Next.js completa:

```powershell
# Instalar todas as dependências
npm install

# Rodar em modo desenvolvimento
npm run dev
```

Depois acesse: `http://localhost:3000`
